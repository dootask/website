# 聊天待办

在日常沟通中，重要的消息很容易被后续的聊天淹没。**DooTask** 的「聊天待办」功能可以把任意一条聊天消息标记为待办事项，并指定到具体成员，确保关键事项被及时跟进、不被遗漏。

::: tip 🐬 适用场景
- 把群聊中临时分配的事项标记给具体的人；
- 给自己设置一条「稍后处理」的提醒；
- 管理员协助为团队成员分配或清理待办。
:::

## 设置待办

1. 在聊天会话中，找到需要跟进的消息。
2. **右键点击**（移动端长按）该消息，在菜单中选择【设待办】。
3. 选择需要负责该待办的成员（可多选），确认后即完成设置。

被设为待办的成员，会在其待办列表中看到这条消息，方便集中处理。

## 设置提醒时间

为待办设置提醒时间后，到点时系统会**引用原消息**并提醒相关人员，避免错过重要事项。

1. 在待办条目上点击【提醒时间】。
2. 选择具体的提醒日期与时间。
3. 保存后，待办会显示对应的提醒时间；到达设定时间时，相关人员会收到带原消息引用的提醒。

> 清空提醒时间即表示取消该条待办的提醒。

## 取消待办

- 当事项处理完成后，在待办条目上选择【取消待办】即可。
- 系统会弹出确认提示，确认后该待办将从列表中移除。

## 为他人设置 / 取消待办

是否允许为**他人**设置或取消待办，由系统设置中的 **[待办设置权限](/functions/account/system-setting)** 控制：

- **允许**：所有成员均可设置 / 取消他人的待办。
- **禁止**（默认更安全）：仅以下角色可以为他人设置 / 取消待办：
  - 待办本人；
  - 系统管理员；
  - 群主（含群管理员）；
  - 项目负责人（含项目管理员）；
  - 任务负责人。

::: tip 提示
系统管理员可在任意群组中为成员设置或取消待办，便于统一协调团队任务。
:::
