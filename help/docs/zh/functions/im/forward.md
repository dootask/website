# 消息转发

在团队沟通中，经常需要把一段聊天内容分享给其他人或群组。**DooTask** 支持单条转发与**合并转发**，让分享聊天内容更方便、更有条理。

## 单条转发

1. 在聊天中，右键点击（移动端长按）需要分享的消息。
2. 选择【转发】。
3. 选择目标会话（成员或群组）并确认发送。

## 合并转发

当需要一次性分享多条消息时，可使用**合并转发**，将多条消息打包成一条记录转发出去，接收方点击即可查看完整的聊天上下文。

1. 在聊天中进入**多选模式**（长按消息或通过菜单进入）。
2. 勾选需要分享的多条消息。
3. 选择【合并转发】，再选择目标会话并确认。

::: tip 适用场景
- 把一段讨论的来龙去脉完整转交给相关同事；
- 将群内的关键决策汇总后转发给上级或其他群组；
- 避免逐条转发造成的刷屏和上下文割裂。
:::
